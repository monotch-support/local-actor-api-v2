    Local Actor API v2 2.0.0

    Local Actor API for managing certificates, subscriptions, deliveries, and capabilities. This API provides endpoints for actor-based certificate management and message routing.

    Contact: API Support support@example.com
    License: MIT

    Servers

    Description URL
    Local development server http://localhost

    Certificates

    POST /{actorCommonName}/x509/csr

    Sign Certificate Signing Request

    Description

    Submits a Certificate Signing Request (CSR) for signing and returns the signed certificate chain. The response includes the client certificate, intermediate certificates, and root certificate.

    Input parameters

    Parameter In Type Default Nullable Description
    actorCommonName path string No The common name of the actor

    Request body

    {
    "csr": "LS0tLS1CRUdJTiBDRVJUSUZJQ0FURSBSRVFVRVNULS0tLS0K..."
}
    Schema of the request body 
    {
    "type": "object",
    "properties": {
        "csr": {
            "type": "string",
            "description": "Base64 encoded PEM Certificate Signing Request",
            "example": "LS0tLS1CRUdJTiBDRVJUSUZJQ0FURSBSRVFVRVNULS0tLS0K..."
        }
    }
}

    Response 200 OK 

    {
    "certificates": [
        "LS0tLS1CRUdJTiBDRVJUSUZJQ0FURS0tLS0t...",
        "LS0tLS1CRUdJTiBDRVJUSUZJQ0FURS0tLS0t..."
    ]
}
    Schema of the response body 
    {
    "type": "object",
    "properties": {
        "certificates": {
            "type": "array",
            "description": "Array of base64 encoded PEM certificates. The first element contains \nthe client certificate, followed by intermediate certificates, and \nfinally the root certificate.\n",
            "items": {
                "type": "string"
            },
            "example": [
                "LS0tLS1CRUdJTiBDRVJUSUZJQ0FURS0tLS0t...",
                "LS0tLS1CRUdJTiBDRVJUSUZJQ0FURS0tLS0t..."
            ]
        }
    }
}

    Response 400 Bad Request

    Refer to the common response description: BadRequest.

    Response 404 Not Found

    Refer to the common response description: NotFound.

    Response 500 Internal Server Error

    Refer to the common response description: InternalServerError.
